# pkw-server

Server for PKWrapped.

## Setup

1. Setup dependencies with `uv sync`
2. Grab metadata with `make fetch`.
3. Ensure a `PKBridge` binary is available in the root folder. The binary can be generated with the `Makefile` in the [PKBridge](https://github.com/danielkhir/PKBridge) repository.
4. Launch with `make run`.
